# GCoder-ChatGPT Assistant

> 一款基于 OpenAI Chat API 的助手项目

## 部署及使用

1. `在config_template.js`文件中输入 `OpenAI` 的 `Api Key` 和所需要使用的模型（推荐默认）
2. 将 `config_template.js` 改名为 `config.js`
3. 使用 `Docker` 进行编译
4. `Docker` 运行
